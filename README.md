# StackBoost - for SupportCandy

This plugin is an addon for the SupportCandy help desk system. It provides a number of quality-of-life enhancements and new features to make managing your support system easier and more efficient.

## Features

*   **QOL Enhancements:** A collection of small tweaks to the SupportCandy interface to improve usability.
*   **After Hours Notice:** Display a notice to customers when they create a ticket outside of your normal business hours. **New:** Now features an enhanced rich text editor for customizing the notice message.
*   **Conditional Views:** Show or hide ticket fields based on the value of other fields.
*   **Queue Macro:** Quickly apply a set of actions to a ticket.
*   **After Ticket Survey:** Send a survey to customers after their ticket is closed.
*   **Company Directory:** A fully integrated module for managing and displaying a staff directory.

## Company Directory Module

The Company Directory module allows you to manage a complete staff directory within StackBoost.

*   **Centralized Management:** Manage staff, locations, and departments from a single, tabbed interface within the StackBoost menu.
*   **CSV Import:** Bulk-import new staff members from a CSV file.
*   **Front-End Display:** Display the directory on any page or post using the `[stackboost_directory]` shortcode.
*   **Full Trash Management:** Full support for trashing, restoring, and permanently deleting entries.

## Installation

1.  Upload the `stackboost-for-supportcandy` directory to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Navigate to the **StackBoost** menu in your WordPress admin sidebar to configure the settings.
